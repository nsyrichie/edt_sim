# Texture Files

Place 2D texture assets here.

## Supported Formats
- PNG (preferred, lossless)
- JPG (for photos)
- TGA (for masks)
- EXR (for HDR)

## Naming Convention
- `{type}_{name}_{map}.{ext}`
- Examples:
  - `diffuse_wood_oak.png`
  - `normal_brick_wall.png`
  - `roughness_metal_steel.png`

## Texture Guidelines
- Power of two dimensions (512, 1024, 2048)
- sRGB for color, linear for data maps
- Include metadata in README for each set
