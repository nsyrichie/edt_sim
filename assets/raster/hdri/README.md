# HDRI Environment Maps

Place HDR environment maps here.

## Supported Formats
- `.hdr` (Radiance HDR)
- `.exr` (OpenEXR)

## Naming Convention
- `hdri_{environment}_v{version}.hdr`
- Examples: `hdri_studio_v1.hdr`, `hdri_forest_v2.hdr`

## Usage
These files are used for:
- Environment lighting
- Reflections
- Background images

## Recommendations
- Use 2K resolution for development
- Use 4K+ for final renders
- Include preview image in JPG format
