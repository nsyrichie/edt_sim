# Material Files

Place material libraries and textures here.

## File Types
- `.blend` files with material libraries
- `.png`, `.jpg` texture files
- `.hdr` environment maps

## Naming Convention
- `material_{type}_v{version}.blend`
- `texture_{name}_diffuse.png`
- `texture_{name}_normal.png`

## Material Properties
- PBR workflow preferred
- Include roughness, metallic maps when needed
- Keep texture sizes power of two (1024, 2048)
